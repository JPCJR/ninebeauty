## 🇧🇷 Sobre o Projeto
O **Nine Beauty** é uma landing page desenvolvida para apresentar serviços de beleza de forma moderna, simples e visualmente agradável.

O projeto simula a presença online de um pequeno negócio, com foco em organização de layout, responsividade e experiência do usuário.

Foi criado como parte do meu processo de aprendizado em desenvolvimento web, buscando evolução prática e consistência.

---

## 🇺🇸 About the Project
**Nine Beauty** is a frontend landing page designed to showcase beauty services in a clean, modern, and visually appealing way.

This project simulates how a small business could present its services online, focusing on layout structure, responsiveness, and user experience.

It was built as part of my journey into web development, aiming for practical growth and consistency.
